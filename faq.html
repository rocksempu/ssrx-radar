❓ FAQ — SSRX Event Radar

🔔 Não recebi a notificação

  • Você clicou em Ativar alertas dentro do app?
  • Você instalou o Radar como aplicativo?
  • Você permitiu notificações no navegador?
  • Você limpou dados do navegador recentemente?
Se limpou dados ou desinstalou o app, precisa ativar novamente.

💻 Preciso deixar o site aberto?

Não. O sistema funciona no servidor. Mesmo com tudo fechado, a notificação chega.

📱 Funciona no iPhone?

Sim. Mas apenas pelo Safari.

Abra o link no Safari → Compartilhar → Adicionar à Tela de Início → Abra o app e ative os alertas.

🤖 Funciona no Android?

Sim. Use o Chrome.

Menu ⋮ → Adicionar à tela inicial → Abra o app e ative os alertas.

🖥️ Funciona no PC?

Sim. Use Chrome ou Edge.

Clique em Instalar como aplicativo e depois ative os alertas.

❗ Recebi várias notificações do mesmo evento

Isso acontecia nas versões antigas. Hoje o sistema envia apenas uma notificação por evento.

🗓️ Como os horários funcionam?

Todos os horários seguem horário do Brasil (GMT-3).

⚙️ Como adiciono ou altero eventos?

Basta editar o arquivo events.json. O radar atualiza automaticamente.

🧹 Limpei dados do navegador e parou de funcionar

É esperado. Abra o app novamente e clique em Ativar alertas.

🚀 Por que isso funciona mesmo com tudo fechado?

Porque o alerta vem do servidor do Radar, não do seu navegador.
